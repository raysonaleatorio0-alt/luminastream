# **App Name**: LuminaStream

## Core Features:

- Intelligent Discovery Tool: A generative AI-powered tool that analyzes your mood to suggest specific movies and series, utilizing reasoning to recommend the perfect match from the TMDB database.
- TMDB Global Sync: Seamless integration with the TMDB API (Key: 39613603847eace58c510bb0a4e43856) to fetch high-resolution posters, detailed synopses, and real-time cast information.
- Universal Omni-Player: Advanced video player component capable of streaming multiple formats including Master .txt manifests and high-bitrate video streams.
- Dynamic Catalog UI: A modern, edge-to-edge browsing interface that displays movies and series in an immersive grid layout optimized for discovery.
- Cross-Platform Watchlist: A simple client-side saving mechanism to track and organize your personal queue of upcoming content.

## Style Guidelines:

- Primary Color: Vibrant Electric Violet (#906BFF) to evoke a sense of digital magic and premium entertainment.
- Background Color: Deep Charcoal Abyss (#110F14), a desaturated variant of the primary hue for a high-contrast dark mode experience.
- Accent Color: Ethereal Periwinkle (#AFBCFF) for highlight states and primary action buttons to ensure visibility against dark tones.
- Headline and UI Font: 'Space Grotesk' for a technical, modern look. Body text font: 'Inter' for maximum readability across metadata blocks.
- A sophisticated layout featuring oversized hero banners, rounded content cards, and hidden sidebars to focus purely on the visual assets.
- Thinline glassmorphic icons that maintain the modern minimalist aesthetic without cluttering the UI.
- Slow, buttery smooth scale transitions upon hovering content cards and micro-animations for player controls.
